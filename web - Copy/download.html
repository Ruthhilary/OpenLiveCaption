Quick Start Guide

  1. Download the installer for your platform
  2. Install OpenLiveCaption (takes 1 minute)
  3. Launch the application
  4. Select your audio source (microphone or system audio)
  5. Click "Start Captions"
  6. Enjoy real-time captions on top of any application!

First launch: The Whisper model will download automatically (1-2 minutes, one-time only)
